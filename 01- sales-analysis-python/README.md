# Sales Analysis using Python

## Objective
To analyze a sample kaggle sales dataset and generate business insights.

## Tools Used
- Python
- Pandas
- Matplotlib

## Steps Performed
1. Loaded CSV data
2. Cleaned data by removing duplicates and null values
3. Calculated region-wise sales
4. Identified top 10 products
5. Built monthly sales trend
6. Exported outputs and charts

## Key Insights
- East and West regions contributed strongly to sales
- Canon imageClass 2200 Advanced Copier products generated high Sales
- Monthly sales trend helps identify growth periods
